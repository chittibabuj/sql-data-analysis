# Heart Disease Data Analysis

A comprehensive SQL and Python analysis of the Heart Disease UCI dataset.

## Project Overview

This project demonstrates data analysis skills using SQL queries to extract insights from medical data, followed by Python visualization and statistical analysis.

## Dataset

- *Source:* Heart Disease UCI Dataset (Kaggle)
- *Records:* 303 patients
- *Features:* 14 medical attributes
- *Target:* Heart disease diagnosis (0 = No, 1 = Yes)

## Files

- queries.sql - 10 SQL queries for data analysis
- analysis.py - Python script to execute queries and generate visualizations
- heart_disease.csv - Dataset
- analysis_results.png - Visualization outputs

## Key Findings

- Total patients analyzed: 303
- Patients with heart disease: ~54%
- Average age: ~54 years
- Gender distribution analysis included
- Correlation between medical metrics and disease status

## SQL Queries Covered

1. Overview statistics
2. Age analysis
3. Gender distribution
4. Cholesterol levels
5. Blood pressure analysis
6. Age group risk assessment
7. Chest pain types
8. High-risk patient identification
9. Maximum heart rate analysis
10. Comprehensive summary statistics

## Technologies Used

- SQL (SQLite)
- Python (pandas, matplotlib, seaborn)
- Data visualization
- Statistical analysis
